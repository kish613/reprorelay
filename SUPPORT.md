# Support

Use the following channel for the type of help you need:

- **Bug reports:** open a GitHub issue using the bug form and include a minimal reproduction with secrets and customer data removed.
- **Feature requests:** use the feature form or start a Discussion for early product ideas.
- **Setup questions:** use GitHub Discussions and include your deployment model, Node version, storage driver, and relevant redacted logs.
- **Security vulnerabilities:** follow [SECURITY.md](SECURITY.md) and report privately.

This project does not operate a shared hosted backend. Questions about a third-party or company-hosted ReproRelay deployment should be sent to that deployment's operator.

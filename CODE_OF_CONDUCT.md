# Code of Conduct

## Our standard

We want ReproRelay to be a welcoming, practical project. Be respectful, assume good intent, give actionable technical feedback, and avoid harassment, discrimination, personal attacks, doxxing, or publishing another person's credentials or private data.

## Project spaces

This standard applies to issues, pull requests, Discussions, releases, and other spaces where someone represents the ReproRelay project. Technical disagreement is welcome; abusive behaviour is not.

## Enforcement

Maintainers may edit or remove content, reject contributions, or temporarily or permanently restrict participation when behaviour threatens people, project safety, or productive collaboration. Sensitive conduct concerns should be raised privately through the repository owner's GitHub profile rather than posted publicly.
